# Counter App

A simple web counter app using Web Assembly with RUST & YEW

Tutorial from [Let's Get Rusty](https://youtu.be/KmOeFrwz8BM).

## Setup

You need RUST >= 1.56.0

```bash
$ rustup target add wasm32-unknown-unknown
```

```bash
$ cargo install trunk
```

```bash
$ cargo build
```

```bash
$ trunk serve
```
